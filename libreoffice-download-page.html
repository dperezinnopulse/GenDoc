Operating Systems

LibreOffice 25.2.5 is available for the following operating systems/architectures:

  • Linux x64 (deb)
  • Linux x64 (rpm)
  • macOS (Aarch64/Apple Silicon)
  • macOS x86_64 (10.15 or newer required)
  • Windows
  • Windows Aarch64
  • Windows x86_64 (Windows 7 or newer required)
